# PromotionAI - 智能促销系统

**PromotionAI** 是一个基于AI技术的智能内容营销平台，专注于财经/健康领域的资讯采集、智能处理、多渠道分发和效果追踪。

## 项目简介

PromotionAI利用先进的AI技术，构建了一个完整的财经/健康资讯采集、AI处理、多渠道分发和数据分析的闭环系统，实现从内容采集到转化分析的全流程自动化。

## 核心能力

### 1. 智能内容生成
- **AI文案生成**: 基于大模型的智能文案生成
- **多风格适配**: 焦虑型、干货型、故事型等多种风格
- **A/B测试**: 自动化A/B测试，优选最佳内容
- **质量评估**: 多维度质量评分体系

### 2. 人机协同与反馈闭环
- **人工复核**: 关键数据人工确认机制
- **运营反馈**: 评分、修正、建议反馈闭环
- **模型优化**: 基于反馈持续优化AI模型
- **爆款学习**: AI学习"爆款"内容规律

### 3. 全链路监控
- **实时监控**: 采集、生成、分发全流程监控
- **可视化看板**: 内容流转状态实时展示
- **性能指标**: 系统性能和业务指标监控
- **告警通知**: 钉钉/飞书机器人自动告警

### 4. 安全合规
- **内容审核**: 敏感词过滤和违禁词库
- **AI标识**: 明确标注AI生成内容
- **数据脱敏**: 用户隐私保护
- **合规审计**: 定期合规性检查

### 5. 容灾备份
- **服务降级**: AI服务不可用时的降级策略
- **数据备份**: 多层级数据备份与恢复
- **容灾演练**: 定期容灾演练与预案验证
- **高可用性**: 99.5%以上系统可用性

## 技术架构

- **后端**: Node.js + Koa / Python + FastAPI
- **数据库**: MySQL + Redis + MongoDB + 向量数据库
- **AI服务**: OpenAI API / 本地模型
- **前端**: Vue3 + Element Plus
- **监控**: Prometheus + Grafana
- **安全服务**: 阿里云内容安全 / 腾讯云文本检测
