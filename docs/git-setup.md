## 常用 Git 命令

### 基本操作
```bash
# 查看状态
git status

# 添加文件
git add .

# 提交更改
git commit -m "提交信息"

# 推送到远程
git push

# 从远程拉取
git pull
```

### 分支操作
```bash
# 查看分支
git branch

# 创建新分支
git checkout -b 新分支名

# 切换分支
git checkout 分支名

# 合并分支
git merge 分支名
```

### 远程操作
```bash
# 查看远程仓库
git remote -v

# 推送当前分支到远程
git push -u origin 分支名

# 拉取远程更新
git pull origin 分支名
```
