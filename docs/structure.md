# PromotionAI 项目结构说明

## 项目概述
PromotionAI 是一个基于AI技术的智能内容营销平台，专注于财经/健康领域的资讯采集、智能处理、多渠道分发和效果追踪。

- `financial-health-content-system.md` - 主需求方案文档
- `overview.md` - 项目概览文档
- `complete-feature-overview.md` - 完整功能概览
- `project-implementation-summary.md` - 项目实施总览
