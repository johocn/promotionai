# PromotionAI 项目

**PromotionAI** 是一个基于AI技术的智能内容营销平台，专注于财经/健康领域的资讯采集、智能处理、多渠道分发和效果追踪。

## Git 仓库信息

- **远程仓库**: git@github.com:johocn/promotionai.git
- **分支**: master

## 目录结构

- `docs/` - 项目文档
- `src/` - 源代码
- `tests/` - 测试文件
- `config/` - 配置文件
- `data/` - 数据文件
- `logs/` - 日志文件
